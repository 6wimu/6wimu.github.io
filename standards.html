Section 02

Relevant Metadata Standards

Standards and semantic models used directly or as design references.

Overview Implementation note

Standards overview

Dublin Core TermsUsed directly for record identifiers and descriptive naming rationales. It offers broadly reusable semantic properties across domains.
Schema.orgInformed earlier treatment of names and nationality. The final XSD uses local elements where direct XSD implementation is impractical.
FOAFInformed the distinction between family name and given name in person-name modeling.
SKOSInformed definition writing and the structure of controlled vocabularies.
ISO 3166-1Provides two-letter nationality codes such as KR, US, GB, and JP.
IPAProvides an optional pronunciation layer beyond Pinyin for precise phonetic description.

Implementation note

This schema is an application profile: it combines external semantic standards with locally defined elements created for the Chinese-name recommendation domain. Not every referenced standard is imported as an XSD. Some standards guide meaning, naming, and interoperability, while the final technical implementation prioritizes reliable schema validation.
