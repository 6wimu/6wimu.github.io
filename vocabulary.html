dcterms:identifier — Record ID

Source: Dublin Core Terms

Definition: Unique identifier assigned to each record.

Data value: string

Cardinality: Required · single

Example: CN-2026-0002

originalName — Original Name

Source: Local

Definition: The person’s full name before Chinese-name creation.

Data value: string

Cardinality: Required · single

Example: David

inLanguage — Original Name Language

Source: Local controlled vocabulary

Definition: Language context of the original name.

Data value: English / Korean / Japanese / Other

Cardinality: Required · single

Example: English

familyName — Chinese Family Name

Source: Local; informed by FOAF

Definition: Chinese surname selected for the created name.

Data value: string

Cardinality: Optional · single

Example: 戴

givenName — Chinese Given Name

Source: Local; informed by FOAF

Definition: Chinese given-name characters excluding the surname.

Data value: string

Cardinality: Required · single

Example: 伟

fullChineseName — Full Chinese Name

Source: Local

Definition: Complete Chinese name used for display and retrieval.

Data value: string

Cardinality: Required · single

Example: 戴伟

pinyin — Pinyin

Source: Local

Definition: Standard Hanyu Pinyin with tone marks.

Data value: string

Cardinality: Required · single

Example: Dài Wěi

ipa — IPA Pronunciation

Source: Local

Definition: Pronunciation represented with IPA.

Data value: string

Cardinality: Optional · single

Example: [taɪ̯˥˧ weɪ̯˨˩˦]

nationality — Nationality

Source: Local; ISO 3166-1 values

Definition: Applicant nationality expressed as a two-letter country code.

Data value: [A-Z]{2}

Cardinality: Required · single

Example: US

mappingType — Name Mapping Type

Source: Local controlled vocabulary

Definition: Method used to transform the original name into a Chinese name.

Data value: Phonetic / Semantic / Localization / HanjaSphere / Phono-Semantic

Cardinality: Required · single

Example: Localization

dcterms:description — Name Meaning and Rationale

Source: Dublin Core Terms

Definition: Meaning of the selected characters and the design intention.

Data value: string

Cardinality: Required · single

Example: Design explanation

hanziRadical — Hanzi Radical

Source: Local

Definition: Radical information for important characters.

Data value: string

Cardinality: Optional · repeatable

Example: 亻

pronunciationDifficulty — Pronunciation Difficulty

Source: Local controlled vocabulary

Definition: Estimated difficulty for non-Chinese speakers.

Data value: Low / Medium / High

Cardinality: Optional · single

Example: Low

culturalElement — Cultural Element

Source: Local controlled vocabulary

Definition: Cultural or philosophical consideration used in the name.

Data value: Zodiac / FiveElements / None

Cardinality: Optional · repeatable

Example: None
